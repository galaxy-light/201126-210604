임베디드 애플리케이션 구현
- 예외처리
- 제네릭 클래스 : 모든 타입 사용 가능
- 컬렉션 클래스 : 자료 구조(데이터베이스의 엔진 역할)
- I/O 스트림
- 쓰레드
- DB(데이터베이스)
- Network
- UI framework


* 프로세스 : 실행중인 프로그램
프로그램 안에는 쓰레드가 존재, 이때 쓰레드를 메인 쓰레드라고 부름
하나의 프로세스는 하나의 메인 쓰레드를 가짐
하나의 프로세스안에는 작업 쓰레드(work thread) 또한 있지만 작업 쓰레드는 n개가 있을 수 있음
작업 쓰레드가 여러개 있는 것을 멀티 쓰레드(multi thread)라고 부름
