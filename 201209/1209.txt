자바는 카멜 스타일
자바에서는 첫자가 대문자면 클래스

타입은 하나의 클래스에서 동일하게, 더블이면 더블 플롯이면 플롯 하나로 통일해서 사용

* 제어문(if) // if <- 소문자, if문에서 가장 중요한 것은 조건, {}안에 여러줄 가능
 1. if(조건) - 조건 1개
if(조건) { 조건이 참이면 중괄호 실행
	실행문;
}

 2. if(조건) ~ else - 조건 2개
if(조건) {조건이 참이면 실행문1 실행
	실행문1;
}
else{조건이 거짓이면 실행문2 실행
	실행문;
}

3. if(조건) - 조건 n개
if(조건) {참을 찾을 때까지 실행, 내려가면서 참을 찾으면 실행 후 맨 밑으로 내려감(빠져나감)
	실행문1;
}
else if(조건) {
	실행문2;
}
else if(조건) {
	실행문3;
}
else{생략가능, 모든 조건이 거짓이면 else실행, 프로그램의 목적에 따라 else 필요/불필요
	실행문;
}
